AudioTuner Desktop v1.0.0
=============================

智能音频调音工具桌面版

功能特性:
- 智能音频风格匹配
- Adobe Audition集成
- 批量音频处理
- 多格式音频转换
- 音频质量评估
- 实时性能监控

系统要求:
- Windows 7/8/10/11 (64位)
- 至少2GB RAM
- 500MB可用磁盘空间

使用说明:
1. 双击桌面图标启动应用
2. 首次运行会在用户目录创建配置文件夹
3. 支持拖拽音频文件进行处理
